Installation Paths (after running install.sh):

- Configuration: /etc/slurm-usage-history-exporter/config.json
- Service: /etc/systemd/system/slurm-usage-history-exporter.service
- Timer: /etc/systemd/system/slurm-usage-history-exporter.timer

Key Commands:
- Install: sudo ./install.sh
- Configure: sudo nano /etc/slurm-usage-history-exporter/config.json
- Test: sudo slurm-usage-history-exporter --dry-run --verbose
- Run: sudo slurm-usage-history-exporter
- Enable timer: sudo systemctl enable slurm-usage-history-exporter.timer
- Start timer: sudo systemctl start slurm-usage-history-exporter.timer
- Check status: systemctl status slurm-usage-history-exporter.timer
- View logs: journalctl -u slurm-usage-history-exporter.service -f
